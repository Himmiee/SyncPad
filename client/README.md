# SyncPad Frontend

The frontend for SyncPad — a realtime collaborative learning space built with Vue.js and Tailwind CSS.

## 🛠 Tech Stack

- Vue.js
- Vite
- Tailwind CSS
- TypeScript

## 🧪 Running Locally

### 1. Install dependencies

```bash
cd client
npm install
```

### 2. Start the dev server

```bash
npm run dev
```
